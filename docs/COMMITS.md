# Guia de Commits

Este projeto utiliza o padrão **Conventional Commits 1.0.0**.

## Estrutura

<tipo>(escopo opcional): descrição

Exemplos:

feat(auth): adiciona autenticação JWT

fix(upload): corrige erro no envio de imagens

docs(readme): atualiza instruções de instalação

refactor(api): simplifica serviço de brinquedos

style(ui): ajusta espaçamento dos cards

test(events): adiciona testes do módulo de eventos

chore(deps): atualiza dependências

ci(github): adiciona workflow de deploy

perf(images): otimiza carregamento das imagens

build(docker): ajusta imagem do backend

revert: remove integração antiga do upload
