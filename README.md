# 🎪 Site Institucional - Empresa de Eventos e Locação de Brinquedos

## Sobre

Este projeto consiste em uma plataforma completa para empresas de recreação, locação de brinquedos e realização de eventos.

O sistema possui um site institucional moderno, responsivo e otimizado para SEO, além de um CMS (Painel Administrativo) que permite gerenciar todo o conteúdo sem necessidade de alterar código.

---

## Objetivos

- Site institucional moderno
- SEO otimizado
- Alta performance
- Painel administrativo completo
- Gerenciamento de conteúdo
- Gerenciamento de imagens
- Totalmente responsivo
- Arquitetura escalável
